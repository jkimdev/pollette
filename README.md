# POLLETTE - 랜덤 당첨자 뽑기

## 앱 개요
점심 쏘기, 커피 내기, 우주적 확률로 결정!
뉴턴의 중력 법칙을 활용한 초과학적 뽑기 게임!
이제 점심을 누가 살지, 커피를 누가 쏠지 우주의 질서에 맡겨보세요.
무려 137억 년의 우주 역사가 당신의 운명을 결정할 겁니다.
별들이 춤추고, 행성이 끌어당기는 이 신비로운 순간을 놓치지 마세요!

## 주요 기능
- **지구**: 창백한 푸른 점.
- **달**: FLY ME TO THE MOON.
- **무스타파**: IT'S OVER, ANAKIN!
- **천왕성**: 천왕성, 얼음 거인.

## 개발 환경
- **언어**: SwiftUI
- **환경**: SpriteKit
